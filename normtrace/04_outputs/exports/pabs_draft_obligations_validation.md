## 9. Limitations of the Draft Text Extraction

### 9.1 Source file quality
The source file `draft_pabs_annex_bureau_9_march_2026.md` was generated from PDF text extraction (noted in the file header). Manual review is still recommended for footnotes, line breaks and any bracketed negotiation text that may not have been fully captured in the markdown conversion.

### 9.2 Bracketed negotiation text
The source file notes that the highlighting and bracketing conventions used in the IGWG context follow those established at IGWG meetings. Some bracketed options or alternative formulations visible in the original PDF negotiation document may not be fully represented in the markdown extraction. Accordingly, Tabla B rows should be understood as reflecting the main Bureau-proposed text, not necessarily all bracketed alternatives.

### 9.3 'As mutually agreed' qualifiers
Several rows (SECIIC_010, SECIIC_008) involve technology transfer "as mutually agreed" — this qualifier is significant because it preserves the voluntary nature of technology transfer and is replicated from the adopted Agreement (footnote 8). Rows involving this qualifier have been coded conservatively, reflecting the negotiated and non-mandatory character of the corresponding obligations.

### 9.4 Annual monetary contributions criteria
The criteria for determining annual monetary contributions (SECIIC_006) — described in the draft as 'objective, transparent and measurable' — are not defined in the Bureau draft. This is one of the most contested elements in the PABS negotiations. The row reflects what the draft text proposes but the actual amounts, formulas and payment mechanisms remain entirely to be determined.

### 9.5 'Pathogen with pandemic potential' scope
The definition in SECIB_001 is a key unresolved element. The two-element conjunctive definition (sustained transmission potential + capacity to cause pandemic emergency) reflects the Bureau draft's approach, but the scope of this definition relative to the existing Pandemic Influenza Preparedness (PIP) Framework and the Nagoya Protocol is one of the most technically and politically contested aspects of the PABS negotiations. This definition determines the entire coverage of the PABS System.
